# workouts-api

```
As a user,
So my details are protected,
I'd like to log in to my profile.
```

```
As a user,
So I know my routine for the day,
I'd like to see my workout on the screen.
```

```
As a user,
So I can obtain a more rounded approach,
I'd like to categorise my workouts by muscle group.
```

```
As a user,
So I can keep track of my workouts,
I'd like to have the option to mark it as completed.
```

```
As a user,
So I don't exhaust one muscle group,
I'd like the workout to change after I mark it completed.
```

```
As a user,
So I can plan ahead,
I'd like to see my schedule for the week in a readable format.
```

```
As a user,
So I can switch it up from time to time,
I'd like the workout types to change each week (e.g. high/low reps).
```
